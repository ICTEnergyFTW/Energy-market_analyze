# Energy-market_analyze
Data analysis and presentation plots from the energy markets based on open API data from data sources such as Entso-e, Fingrid, Ember.

ENTSO-E (Transparency Platform / API)
Many datasets are available for free re-use under CC BY 4.0 (or without needing prior approval from primary owners for listed "Open Data"). 
transparencyplatform.zendesk.com

Requirements: Mention the ENTSO-E Transparency Platform as the source (in accordance with good industry practices). Do not imply endorsement or sponsorship. Check the current list of data open for re-use, as some require contacting primary data owners. Attribution to the platform (and ideally the specific dataset/primary owner) is key. Just saying "source: ENTSO-E" in a caption or README is typically enough for plots/figures, but preserve any notices and link back where possible.

Fingrid (Open Data API)
Explicitly licensed under Creative Commons Attribution 4.0 International (CC BY 4.0) for Fingrid-produced datasets (others may vary—check per dataset). 
data.fingrid.fi

Attribution requirements (must preserve copyright notice if present, indicate changes, link to license and original where possible). Recommended example:  "Source: Fingrid / data.fingrid.fi, license CC 4.0 BY". Their general legal terms also apply. Proper credit is mandatory; sharing the source satisfies this for most uses like plots and derived figures.

Ember (Climate/Electricity Data API)
All content and data released under CC BY 4.0. 
ember-energy.org +1

You must credit Ember when sharing/adapting. They appreciate being informed of usage but do not appear to impose strict formatting beyond standard CC BY.
